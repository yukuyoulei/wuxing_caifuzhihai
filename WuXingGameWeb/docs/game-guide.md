# 五行探险记 - 游戏指南

## 游戏简介

五行探险记是一款基于中国传统五行相克原理的策略冒险游戏。玩家通过探险收集金、木、水、火、土五种元素，在随机生成的地图中与对手进行战斗，使用阴阳货币辅助战斗，不断前进冒险。

## 游戏规则

### 初始状态
- 玩家出生在坐标 (0, 0)
- 初始拥有 0 个五行元素
- 初始拥有 0 个阴阳货币
- 出生点有 NPC 可以补充元素（任何元素低于 10 时自动补充至 10）

### 探险机制

#### 移动规则
1. 点击"开始探险"后，系统随机选择四个对角方向之一：
   - 左上（x-1, y-1）
   - 左下（x-1, y+1）
   - 右上（x+1, y-1）
   - 右下（x+1, y+1）

2. 到达新位置后，根据距离出生点的距离生成对手元素

#### 战斗系统

战斗分为 5 个回合，每回合流程如下：

1. **选择元素**：从你的元素库存中选择一种元素
2. **揭示对手**：点击对手的隐藏元素，揭示其类型和数量
3. **克制判定**（受技能影响）：
   - 如果你的元素克制对手元素：对手数量减少（基础50%，受阴属性技能加成）
   - 如果你的元素被对手克制：对手数量增加（基础200%，受阳属性技能减免）
   - 其他情况：对手数量不变
4. **胜负判定**：你的元素数量 > 对手元素数量则本回合胜利

#### 战斗结算

- **胜利条件**：5 回合中至少赢 3 回合
- **冒险成功**：
  - 从你赢的回合中选择一个，获得该回合对手的元素数量
  - 战斗中使用的元素不会被消耗
  - 根据距离获得阴或阳货币
- **冒险失败**：
  - 失去所有失败回合中使用的元素
  - 可以选择使用货币扭转一个回合的结果
  - 也可以选择直接返回出生点 (0, 0)，不使用货币

### 五行相克规则

```
金 → 克制 → 木
木 → 克制 → 土
土 → 克制 → 水
水 → 克制 → 火
火 → 克制 → 金
```

### 货币系统

#### 货币获取
- 每次冒险成功后，根据距离出生点的距离获得随机数量的阴或阳货币
- 每次只能获得一种货币
- 有 30% 概率不获得货币

#### 货币使用
- 当冒险失败时，可以使用货币强行改变一个回合的结果
- 消耗数量 = 该回合你的元素数量与对手元素数量的差值
- 每次冒险失败只能扭转一个回合
- 扭转后重新检查是否达到胜利条件（至少 3 胜）
- 也可以选择不使用货币，直接返回出生点

## 游戏策略

### 元素管理
- 在出生点时，确保所有元素都补充到 10 个
- 冒险成功时不消耗元素，可以放心使用
- 冒险失败时会失去失败回合使用的元素，需要谨慎选择
- 优先使用数量多的元素，减少失败时的损失
- 避免在失败回合中使用稀缺元素

### 战斗技巧
- 观察对手元素的分布，选择克制关系有利的元素
- 记住五行相克规则，利用克制关系减少对手数量
- 优先使用数量多的元素，确保在对比中获胜
- 可以在多个回合中重复使用同一种元素
- 如果预感会失败，尽量使用数量较少的元素减少损失

### 货币策略
- 积累足够的阴阳货币作为保险
- 只在关键时刻使用货币扭转战局
- 如果货币不足或损失不大，可以选择直接返回出生点
- 权衡使用货币的成本与失去元素的损失
- 优先扭转差距较小的回合，节省货币

### 技能系统

#### 技能概述
- 每个五行元素都有对应的技能
- 每个技能有两个升级方向：阴属性和阳属性
- 技能等级从 0 级开始，可以无限升级

#### 阴属性技能（克制加成）
- **效果**：增强你的元素克制对手时的效果
- **基础效果**：减少对手元素 50%
- **升级加成**：每级增加 5% 减少效果
- **示例**：阴属性 3 级时，克制效果为减少对手 65%（50% + 3×5%）

#### 阳属性技能（抗性加成）
- **效果**：减免你的元素被对手克制时的惩罚
- **基础效果**：对手元素增加 200%
- **升级减免**：每级减少 10% 增加效果
- **示例**：阳属性 5 级时，被克制效果为对手增加 150%（200% - 5×10%）

#### 技能升级成本
- **元素消耗**：基础 10 个 + 当前等级 × 5 个
- **货币消耗**：基础 3 个 + 当前等级 × 2 个
- **货币类型**：
  - 升级阴属性需要消耗阴货币
  - 升级阳属性需要消耗阳货币

#### 技能策略
- 优先升级常用元素的技能
- 根据战斗风格选择升级方向：
  - 进攻型：优先升级阴属性，增强克制效果
  - 防守型：优先升级阳属性，减少被克制损失
- 平衡五种元素的技能等级，避免短板
- 合理分配阴阳货币，不要全部用于技能升级

### 探险规划
- 距离出生点越远，对手越强，但奖励也越丰厚
- 适时返回出生点补充元素
- 平衡风险与收益

## 游戏特色

- **传统文化融合**：基于中国传统五行理论设计
- **策略性强**：需要合理规划元素使用和战斗策略
- **随机性高**：每次探险都是全新的挑战
- **进度保存**：游戏进度自动保存在本地
- **视觉精美**：采用五行传统配色，界面美观

## 操作说明

- **开始探险**：点击"开始探险"按钮开始新的冒险
- **选择元素**：点击你的元素图标选择用于战斗的元素
- **揭示对手**：点击对手的问号图标揭示其元素
- **升级技能**：在技能面板中点击"升级"按钮提升技能等级
- **NPC 补充**：在出生点点击"NPC 补充元素"按钮补充元素
- **返回出生点**：点击"返回出生点"按钮回到 (0, 0)
- **重置游戏**：点击"重置游戏"按钮清空所有进度
